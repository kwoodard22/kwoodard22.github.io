PORTFOLIO

Road Trip is an educational game that allows players to test their knowledge of state capitals on a trip across the US. This app was a result of a four person group project.

Constituent Connect seeks to remedy the problem of lost caller data in congressional offices by providing a dashboard to record calls, search bills, collect caller details, and display call data in a chart. This was my final class project.
