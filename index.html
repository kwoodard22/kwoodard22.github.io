PORTFOLIO

Billmate allows roommates to easily split monthly house expenses. This was my first project in my Web Development Immersive program. The technologies used were Ruby, Rails, and Authentication using Devise.

Road Trip is an educational game that allows players to test their knowledge of state capitals on a trip across the US. This app was a result of a four person group project and used Rails, JavaScript, and jQuery.

Constituent Connect seeks to remedy the problem of lost caller data in congressional offices by providing a dashboard to record calls, search bills, and display call data in a chart. This was my final class project, and it uses Rails, Angular.js, and JavaScript.
